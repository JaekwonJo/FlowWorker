# Afterglow Groove 제작 회의록

## 2026-05-07 리브랜딩 회의

### 결정
- 새 프로젝트는 `Afterglow Groove`다.
- 한국어 운영명은 `애프터글로우 그루브`다.
- Codex 별칭은 `루프PD`에서 `그루브PD`로 변경한다.

### 새 방향
- Suno로 원곡을 만든다.
- 음악은 flat melody, no climax, soft falsetto, 70s/80s groove 중심이다.
- 영상은 FlowWorker로 만든 빈티지 시네마틱 뮤직비디오 컷이다.
- 목표 반응은 “음악 좋은데, 영상도 명작 단편처럼 볼만하다”이다.

### 샘플 톤
- `Sample4`: 어두운 아파트/관찰자/누아르
- `Sample5`: 아날로그 스케치/스토리보드
- `Sample6`: 흑백 패션 필름/팝 아이콘
- `Sample7`: 빈티지 자동차/붉은 시트/밤의 여운

### 역할 구조
- 사용자는 방향과 취향을 말한다.
- 그루브PD가 총괄 프로듀서로서 음악 콘셉트, Suno 프롬프트, 가사, 영상 플롯, FlowWorker 프롬프트를 만든다.
- FlowWorker는 실행 직원이다.

### 실행 원칙
- 실제 제작은 오로지 FlowWorker를 통해 진행한다.

## 2026-05-07 서사/컷 연결 원칙 추가 회의

### 결정
- 영상은 가사와 직접 상관없이 독립적인 서사/스토리/플롯을 가진다.
- 곡은 정서와 리듬을 제공하고, 영상은 그 위에 별도의 단편영화를 얹는다.
- 마지막 프레임을 뽑아 다음 영상으로 이어가는 원칙은 유지한다.
- 단, 모든 컷을 무조건 이어가지 않는다.
- 필요할 때는 새 S컷을 만들어 장면을 끊고 새 장소/시간/상태로 전환한다.

### 제작 판단
- 이어가기 컷은 몰입과 그루브를 만든다.
- 새 장면 컷은 환기와 서사를 만든다.
- 두 방식을 섞어야 음악 채널이면서도 단편 뮤직비디오처럼 보인다.
