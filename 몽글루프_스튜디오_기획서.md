# 몽글루프 스튜디오

## 한 줄 정의
Flow/Veo의 시작 프레임 고정과 확장 기능, Suno 음악을 조합해 귀엽고 편안하게 계속 보게 되는 3-4분 롱폼 영상을 만드는 사이드 프로젝트.

## 역할 고정
- 사용자: 원작자, 취향 결정자, 최종 승인자.
- Codex: 총괄 프로듀서/감독. 별칭은 `루프PD`.
- Codex는 프롬프트만 쓰는 역할이 아니라, 콘셉트 개발, 시리즈 방향, 장소 동선, 카메라 언어, 감정선, 반복 행동, 샷 리스트, Flow 프롬프트, 자동화 기능 설계까지 담당한다.
- 목표는 사용자의 영화감독 꿈을 실제 제작 파이프라인으로 바꾸는 것이다.

## 핵심 콘셉트
- 대표 캐릭터 `몽글이`가 한 가지 행동을 아주 천천히 이어간다.
- 먹방, 요리, 여행, 명소 산책, 계절 이벤트로 확장한다.
- 한 장면은 16초 단위로 만들고, 마지막 프레임을 다음 시작 프레임으로 넘겨 장면을 이어간다.
- 완성된 롱폼에서 나중에 좋은 구간을 잘라 9:16 숏츠로 재활용할 수 있다.

## 우선순위
1. 롱폼 3-4분 완성 파이프라인
2. 캐릭터/음식/장소 일관성
3. Suno 음악과 영상 감정선 매칭
4. 병렬 제작으로 하루 여러 편 생산
5. 숏츠 재활용

## 1차 시장 전략
- 운영 언어는 한국어로 시작한다.
- 성공 포맷은 일본 감성/일본 음식/일본 명소를 우선 겨냥한다.
- 영상 자체는 말이 거의 없어도 이해되는 구조로 만든다.
- 한국어 제목에 영어 키워드를 섞고, 설명에는 일본어/영어 한 줄을 보조로 붙일 수 있다.
- 자세한 전략은 `몽글루프_스튜디오_한국어_일본성공전략.md`를 기준으로 한다.

## 첫 시리즈 후보
- 딸기 케이크를 천천히 먹는 귀여운 캐릭터
- 피자를 한 조각씩 먹는 캐릭터
- 맥도날드 햄버거를 조용히 먹는 캐릭터
- 일본 벚꽃 명소를 천천히 여행하는 캐릭터
- 국내 감성 명소를 하루 코스로 여행하는 캐릭터

## 롱폼 제작 루프
1. 대표 시작 이미지 생성
2. Flow에서 8초 영상 생성
3. `확장` 누르기 전 `Veo 3.1 - Fast`로 변경
4. 확장해서 16초 영상 생성
5. 16초 영상 다운로드
6. 마지막 프레임 추출
7. 추출 이미지를 다음 `S###` 시작 프레임으로 사용
8. 다음 미세 행동 프롬프트 입력
9. 12-15번 반복해 3-4분 영상 구성

상세 제작 순서와 실제 사용법은 `몽글루프_스튜디오_제작매뉴얼.md`를 기준으로 한다.

## 먹방형 연출 규칙
- 카메라는 대부분 고정 또는 아주 천천히 움직인다.
- 음식은 한입마다 눈에 보이게 조금씩 줄어야 한다.
- 캐릭터 표정은 크게 흔들지 않고 만족감만 조금씩 바꾼다.
- 입, 손, 숟가락, 포크 움직임은 작고 자연스럽게 유지한다.
- 프롬프트는 과한 액션보다 “천천히 한입 더 먹는다”처럼 작게 쪼갠다.

## 여행형 연출 규칙
- 먼저 동선과 장소 순서를 정한다.
- 각 16초 장면은 한 장소의 한 감정만 맡긴다.
- 캐릭터는 같은 옷/소품/색감을 유지한다.
- 이동 장면은 걷기, 고개 돌리기, 사진 찍기, 가방 잡기처럼 단순 행동으로 연결한다.
- 명소 이름을 직접 때려넣기보다 실제 장소 특징을 시각적으로 설명한다.

## FlowWorker에 필요한 기능
- 마지막 프레임 추출: 구현 시작, `flow_worker/video_frame_tools.py`
- UI 버튼: `다음 시작 프레임 추출`
- 다음 단계:
  - 다운로드된 16초 영상 자동 감지
  - 마지막 프레임을 자동으로 다음 `S###.png` 저장
  - Flow 에셋 업로드 자동화
  - 시작 프레임 지정 자동화
  - `확장` 버튼 자동화
  - `Veo 3.1 - Fast` 모델 자동 선택
  - 여러 워커 병렬 큐 설계

## 병렬 제작 방향
- 사용자는 “오늘 영상 몇 개 만들어주세요”라고 요청할 수 있어야 한다.
- Codex 총괄 프로듀서가 기획/설계/프롬프트/워커 배정을 정리한다.
- Flow Worker1, Flow Worker2, Flow Worker3가 각자 다른 영상 또는 테스트를 동시에 담당한다.
- 자세한 병렬 제작 운영은 `몽글루프_스튜디오_병렬제작_설계.md`를 기준으로 한다.
