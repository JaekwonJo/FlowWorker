# FlowWorker 코덱스 설명서

## 목표

- 이 저장소는 `Flow Classic Plus`의 잔재를 끌고 가지 않는 새 독립 앱이다.
- UI는 `Grok Worker`처럼 가볍게 간다.
- 기능은 `Flow` 전용으로만 정리한다.
- 새 Codex가 와도 문서만 보면 바로 이어받을 수 있게 유지한다.

## 꼭 지킬 것

- 코드 수정 시 `README.md`와 `CODEX_연속작업_인수인계_20260304.md`를 같이 갱신한다.
- 기능 단위 작업이 끝나면 가능하면 바로 `git add . -> git commit -> git push`까지 한다.
- `Flow Classic Plus` 코드를 새 앱에서 import 하거나 실행하지 않는다.
- 기존 기능이 필요하면 로직만 참고해서 새 저장소 안에 다시 옮긴다.
- 사용자가 윈도우에서 바로 쓰는 앱이므로, 실행 파일과 실제 사용 흐름을 항상 같이 점검한다.
- `.bat` 파일은 끝에 반드시 `pause`를 넣는다.
- GUI 실행은 가능하면 `.vbs`를 기본으로 유지한다.
- 그록워커와 Edge 디버그 포트/프로필을 절대 공유하지 않는다. FlowWorker 기본값은 전용 프로필과 `9333` 포트를 쓴다.

## 이번 프로젝트의 핵심 방향

- 이미지 / 비디오를 하나의 `Flow Worker`에서 다룬다.
- 불필요한 통합 기능, 이어달리기, 안 쓰는 옵션은 넣지 않는다.
- 사용자가 자주 쓰는 것만 남긴다.

- 2026-04-27: FlowWorker 이미지 모드는 이제 생성 후 카드의 더보기 메뉴를 찾아 다운로드를 시도하고, Playwright download 이벤트를 받아 사용자 저장 폴더로 직접 저장한다.
